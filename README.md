# Tubes-Analisis-Kompleksitas-Algoritma
Metode iteratif dan rekursif sama-sama memiliki kompleksitas waktu O(n) dan menghasilkan output yang sama dalam menghitung deret geometri. Namun, iteratif lebih efisien karena hanya memakai memori O(1) tanpa overhead pemanggilan fungsi. Rekursif membutuhkan memori O(n), lebih lambat, dan berisiko stack overflow pada n besar.
